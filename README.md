# Bank-Marketing-Term-Deposit-Subscription-Prediction
Built a machine learning model to predict term deposit subscriptions using the Bank Marketing dataset. Performed data cleaning, EDA, summary statistics, correlation analysis, and classification modeling with Python, Pandas, Seaborn, Matplotlib, and Scikit-learn.

Project Description

This project focuses on predicting whether a customer will subscribe to a term deposit using the Bank Marketing dataset. The objective was to analyze customer demographics, campaign interactions, and macroeconomic factors to identify the key drivers influencing subscription decisions and build an accurate machine learning classification model.
The project began with data preprocessing, where the dataset was cleaned by checking for missing values, duplicate records, and inconsistencies. Summary statistics were generated to understand the distribution of numerical features, while categorical variables were examined to gain insights into customer characteristics.
An extensive Exploratory Data Analysis (EDA) was conducted using Python libraries such as Pandas, Matplotlib, and Seaborn. Visualizations were created to explore the distribution of the target variable and understand customer subscription patterns. A correlation matrix was used to examine relationships among numerical variables, revealing strong positive correlations between macroeconomic indicators such as Euribor3m, Employment Variation Rate, and Number of Employees, while also identifying moderate negative relationships between campaign-related features.
Following EDA, the data was prepared for machine learning by encoding categorical variables, splitting the dataset into training and testing sets, and applying preprocessing techniques where required. Multiple classification algorithms, including Logistic Regression, K-Nearest Neighbors (KNN), Decision Tree, Random Forest, and Support Vector Machine (SVM), were trained and evaluated to predict whether a customer would subscribe to a term deposit. Model performance was assessed using metrics such as accuracy, confusion matrix, precision, recall, and F1-score.
This project demonstrates practical expertise in data cleaning, exploratory data analysis, feature engineering, data visualization, and predictive modeling. It showcases the application of machine learning techniques to solve a real-world business problem, providing actionable insights that can help banks optimize marketing campaigns, improve customer targeting, and increase subscription rates.

Tools & Technologies: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook.
